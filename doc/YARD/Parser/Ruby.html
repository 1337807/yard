Builds and s-expression by creating AstNode objects with the type provided by the first argument.

Examples:

An implicit list of keywords 

  ast = s(s(:kw, "if"), s(:kw, "else"))
  ast.type # => :list

A method call 

  s(:command, s(:var_ref, "mymethod"))
