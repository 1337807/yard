Public Instance Method Details

format

public Object format(object, format, template)

Formats an object as a specified output format. Default is +:html+.

Parameters:
format - the output format to generate documentation in. Defaults to +:html+, which is a synonym for :xhtml
template - the template sub directory to use, default is :default
